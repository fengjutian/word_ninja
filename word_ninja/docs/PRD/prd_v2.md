# Word Ninja PRD v2.0

## 产品定位

基于 Flutter 全平台开发的 AI 英语学习工具，通过「忍者修炼 + RPG成长 + AI老师」模式游戏化学习。

支持平台：Android / iOS / Windows / macOS / Linux

## 功能模块

- 首页 Dashboard（等级/经验/任务/连续签到）
- 忍者角色系统（Avatar/等级/属性）
- 单词修炼（单词卡/拼写/听音/忍者对战）
- 阅读训练（AI文章/PDF/EPUB/划词翻译）
- 听力训练（精听/听写/跟读）
- 口语训练（AI陪练/发音评分）
- 写作训练（AI范文/IELTS批改）
- AI导师（Sensei Shell 对话）
- 学习计划（AI生成）
- 成就系统（单词/阅读/连续学习）
- 商店系统（Ninja Coin 兑换）
- 排行榜（今日/周/月/好友）
- 数据同步
